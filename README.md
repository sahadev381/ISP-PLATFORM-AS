# ISP Management System

A comprehensive ISP/WISP billing and network management system similar to Splynx.

## Features

- **OLT Management** - BDCOM, Huawei, HSGQ OLTs
- **MikroTik Management** - RouterOS API integration
- **Switch Management** - Cisco, BDCOM, Huawei switches
- **SNMP Monitoring** - Real-time network monitoring
- **Billing Module** - Invoices, payments, subscriptions
- **Hotspot Portal** - SMS OTP, Voucher, PIN authentication
- **Customer Management** - PPPoE, Hotspot users
- **Support Tickets** - Customer support system
- **Leads Management** - Sales pipeline
- **Network Topology** - Visual NOC map
- **Alerts System** - Network故障 notifications

## Tech Stack

- PHP (Native)
- MySQL
- Bootstrap 5
- jQuery
- Chart.js

## Installation

1. Upload to server
2. Import `db.sql` to MySQL
3. Configure `config.php`
4. Login with admin credentials

## Default Login

- Username: admin
- Password: admin
